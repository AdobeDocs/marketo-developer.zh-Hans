# 自助服务流程步骤

借助自助服务流程步骤，集成商可以将Web API部署为Marketo Smart Campaigns中的流程步骤。

流程步骤服务可以：

- 接收商机和相关数据。
- 处理数据，包括调用其他Web服务。
- 通过回调将潜在客户和活动数据写入Marketo。

有关实施详细信息，请参阅GitHub](https://github.com/adobe/Marketo-SSFS-Service-Provider-Interface)上的[开发人员文档。
