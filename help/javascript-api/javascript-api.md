# Javascript API

Marketo客户端JavaScript集成提供了商机跟踪、表单、Web个性化和预测内容功能。 您必须拥有Marketo帐户才能使用这些功能。

实施通常涉及将嵌入代码添加到Web资产。 您还可以调用嵌入代码公开的JavaScript函数以添加功能。

嵌入代码对于Marketo实例是唯一的，因为它包含帐户标识符。 在Marketo用户界面中，转到相应的面板，将嵌入代码复制到剪贴板，然后粘贴到网页中。

## 商机跟踪(Munchkin)

Marketo的[Munchkin JavaScript跟踪代码](lead-tracking.md)通过访问您的网站来生成潜在客户。 它还跟踪未提供个人信息的访客，并创建包括用户的IP地址和其他信息的匿名潜在客户。

在Munchkin页面的Marketo管理区域设置Munchkin。

## Forms 2.0

[Forms 2.0](forms-api-reference.md)允许营销人员在不了解编程的情况下创建Web窗体。 Forms可以位于Marketo登陆页面上，也可以嵌入到您网站的任何页面上。

使用Forms 2.0 JavaScript API扩展Marketo Web窗体的核心功能。

## Web 个性化

[Marketo Web Personalization](web-personalization.md)可帮助您根据潜在客户的身份和行为实时吸引潜在客户。

## 预测性内容

[Marketo预测内容](predictive-content.md)使用机器学习和预测分析向Web访客展示相关内容。 将文本描述和图像添加到您的内容，并在您的网站上嵌入多个内容推荐。
